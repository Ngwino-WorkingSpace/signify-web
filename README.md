
  #  Website for Signify

    SIGNIFY is a simple, community-driven health  data collection and correction platform    designed for African contexts.
     It enables health ministries and organizations to collect accurate, preventive health data using mobile applications and USSD, helping them plan early interventions and save lives.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
